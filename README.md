# blocked-by

> A GitHub App built with [Probot](https://github.com/probot/probot) that A bot that handles \&quot;blocked by\&quot; issue relations.

## Setup

```sh
# Install dependencies
npm install

# Run typescript
npm run build

# Run the bot
npm start
```
